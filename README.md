# ✏ Chatbot Description

본 챗봇은 용종 환자의 생활 식습관 치료 및 솔루션 제공을 위합니다.

- 용종 환자의 첫 병원 내원 시, 생활 식습관 및 기본 정보 조사 진행
- 1년 생활 식습관 기준 솔루션 제공 기능
- 용종 환자의 매 식단 추적 및 솔루션 제공 기능
- 환자별 생활 식습관 저장 및 엑셀 출력 기능
